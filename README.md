# Program WiFi Robot Pengumpan Sampah KRTMI

Deskripsi:
Pengembangan Robot Pengumpan Sampah KRTMI menjadi berbasis WiFi. Project ini menggunakan ESP32 untuk kendali robot dan dilengkapi aplikasi WiFi Remote Control Robot Pengumpan Sampah

## Cara Menggunakan
1. Upload program ke mikrokontroler ESP32 menggunakan Arduino IDE
2. Download Aplikasi Remote Control pada file Remot_Robot_KRTMI.apk
3. Sesuaikan program dengan keperkuan anda
4. Hubungkan robot ke jaringan WiFi sesuai setting di program
